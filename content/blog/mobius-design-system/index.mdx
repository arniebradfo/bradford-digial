This Mobius Design System was created to standardize the UI and general interaction principles for Mobius Software. 
This document acts as a guidebook for design decisions going forward, so these decisions need not be made over and over again. 
The document was made in Adobe Xd, so it can also serve as a copy-paste UI design library for UI components. 
It deals with everything from simple button styles to more complex mapping and vehicle tasking principles.
